# Демонстрация фреймворка для SAGA 
Данный репозиторий агрегирует все микросервисы, которые используются для работы воркфлоу бронирования авиабилетов и отеля. Такой набор сервисов способен показать работу приложения по планированию путешествий, сами сервисы предоставляют минимальное для работы API, каждый сервис обладает swagger документацией, однако их главная цель - это демонстрация работы хореографической, асинхронной SAGA. Каждый сервис ограничен своей доменной областью, а также знает о существовании только своей базы данных, микросервисы могут только получать сообщения через kafka, в остальном они полностью изолированы. 

# Описание сервисов 
flights - сервис, задачей которого является бронирование авиабилетов
hotels - сервис, задачей которого является бронирование отелей
booking - сервис, который хранит пользователей, а также информацию о путешествии, совершает конечную оплату и подтверждение брони, отправляет пользователю уведомление
